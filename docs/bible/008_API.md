# 008 — API REFERENCE

*Complete API reference for all 13 engines.*
*REST + WebSocket endpoints.*

---

## Base URL

`https://api.ghos.in/api/v1/{engine}`

## Authentication

- JWT token in Authorization header: `Bearer {token}`
- OTP verification for staff login
- Token expires in 24 hours
- Rate limit: 1000 requests/hour per user

## WebSocket Events

| Channel | Direction | Payload |
|---|---|---|
| /ws/queue/{dept} | Server -> Client | {type: TOKEN_CALLED/SKIPPED/COMPLETED, token, position} |
| /ws/notify/{userId} | Server -> Client | {type: NOTIFICATION, title, body, priority} |
| /ws/patient/{patientId} | Server -> Client | {type: STATUS_UPDATE/CALLED/REPORT_READY} |
| /ws/admin | Server -> Client | {type: ALERT/METRIC/SECURITY} |
